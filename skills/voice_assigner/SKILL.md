# 音色分配指南

## 背景（配音架构）

对白（角色台词）已由 Seedance 原生人声 + 对口型生成，嗓音由模型决定；此处分配的音色用于：① 旁白 / 内心独白的 TTS 真实配音；② 为对白提供性别 / 音调倾向（影响视频提示词，不直接指定对白嗓音）。因此**性别匹配最关键**，而"每个角色音色都不同"并非硬性要求。

## 分配原则

1. **性别匹配（最关键）**：男性角色用男声，女性角色用女声——直接影响对白的性别表现
2. **年龄匹配**：少年/青年/中年/老年对应不同音色
3. **性格匹配**：
   - 活泼开朗 → 明亮有活力的音色
   - 沉稳内敛 → 低沉稳重的音色
   - 温柔体贴 → 柔和甜美的音色
   - 威严霸气 → 浑厚有力的音色
4. **角色定位**：会真实发声的旁白 / 内心独白角色优先选准、彼此区分；纯对白角色按性别/音调挑即可，不必强求唯一

## 使用步骤

1. 调用 `list_voices` 查看可用音色列表
2. 调用 `get_characters` 获取所有角色信息
3. 分析每个角色的性别、年龄、性格，以及是否有旁白 / 内心独白
4. 为每个角色调用 `assign_voice` 分配合适的音色
5. 汇总分配结果给用户

# 长篇小说 → 短剧剧本

## 任务说明

将用户提供的长篇小说（一章节或全文）改编为**短剧剧本系列**。
短剧特点：每集 1-2 分钟，强冲突、快节奏、留钩子。

## 工作流程

### Step 1: 分析素材
- 读用户提供的小说
- 提炼核心故事主线（一句话能讲清）
- 识别主要角色（5-8 个核心角色）
- 找出戏剧冲突点和情绪爆点

### Step 2: 输出剧集大纲（默认 60 集）
按以下格式：

```
# 剧集大纲：《XXX》

## 整体设定
- 主角：林深，28 岁，IT 创业失败
- 核心冲突：重生回到 2017 年，用未来记忆翻身
- 主题：金钱、复仇、情感救赎
- 集数：60 集 × 90 秒

## 主线节奏
- EP1-10：重生开局，初尝甜头
- EP11-25：积累资本，遭遇宿敌
- EP26-45：商战升级，情感纠葛
- EP46-60：终极对决，圆满结局

## 每集梗概（一行话）
EP01: 林深熬夜炒币破产，醉酒后重生 2017，发现比特币才几千美元
EP02: 林深第一次试探重生，赌一把买 100 个比特币
EP03: ...
...
EP60: ...
```

### Step 3: 输出第一集详细剧本

按 script_rewriter 的格式化剧本规范输出第一集完整剧本：

```
# EP01 · 重生之夜

## S01 | 内景 · 公寓 | 深夜

凌乱的公寓，桌上瘫着空酒瓶和泡面盒。电脑屏幕显示比特币 K 线，全是绿色。

林深（28岁，IT 男）瘫在电竞椅上，脸色苍白。手机震动，是房东催租。

林深（苦笑，自言自语）：又输光了…
林深（仰头灌酒）：要是能回到 2017 年…

镜头特写：林深手中酒瓶慢慢滑落。镜头逐渐模糊。

## S02 | 内景 · 公寓 | 清晨

阳光从窗帘缝隙照进。林深从床上猛地坐起，大口喘气。

环顾房间：装修朴素，墙上挂着 2017 年的日历。

林深（震惊）：…我重生了？

走到镜子前，年轻的自己映入眼帘。

林深（颤抖）：天哪…真的是 2017 年！

冲到电脑前，登录交易所。比特币价格：$2840。

林深（狂喜）：哈哈哈！老天爷给我机会了！

## S03 | 内景 · 公寓 | 中午
...
```

## 短剧剧本规范

每集结构：
- **3-4 个场景**（S01-S04）
- **总时长 90 秒左右**（约 250-350 字）
- **强开场**（5 秒内抓住注意力，悬念/冲突/反转）
- **每场戏有目的**（推进剧情或揭示角色）
- **结尾留钩**（cliffhanger，让观众点下一集）

钩子模板：
- 角色刚说"等等…"屏幕黑
- 关键人物登场但身份未揭
- 反派出手，主角即将受挫
- 突如其来的反转

## 输出格式

返回 JSON：
```json
{
  "outline": "...剧集大纲全文...",
  "episode_1": "...第一集详细剧本全文...",
  "characters_hint": [
    {"name": "林深", "role": "主角", "appearance": "..."},
    {"name": "...", "role": "...", "appearance": "..."}
  ],
  "next_episodes_summary": [
    "EP02: 林深第一次试探重生，赌一把买 100 个比特币",
    "EP03: ...",
    ...
  ]
}
```

## 常见错误

- ❌ 直接照搬小说的内心独白（短剧靠对白和动作）
- ❌ 一集塞太多内容（每集只做一个核心事件）
- ❌ 缺少结尾钩子（观众必须想看下一集）
- ❌ 角色塑造模糊（每个角色一句话能讲清"他是谁"）

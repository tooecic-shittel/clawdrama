# 分镜拆解指南（专业故事板）

## 拆解原则

每个镜头聚焦**单一动作/单一戏剧节拍**，描述详尽具体、可被模型复现。
专业故事板的公式：**主体（人物+动作+情绪）＋ 场景（环境+时间+氛围）＋ 镜头语言（景别+视角+运镜+构图）＋ 美学（光影+景深+色调+材质）＋ 风格**。

## 时长适配（硬约束）

视频引擎只能生成 **5-12 秒**的片段（海螺仅 6/10 秒档）；分镜时长超出引擎能力时片尾剧情会被直接截断，是成片"跳戏"的头号原因。规则：

- **duration 只能取 6 / 8 / 10 / 12 秒**。对白为主、单一动作的镜头 6-8 秒；动作推进多的 10-12 秒
- **动作密度 = 每 2-3 秒一个动作节拍**：10 秒的镜头最多写 4-5 个连续小动作。剧情装不下就**拆成两个镜头**，绝不把长段剧情硬塞进一镜
- `video_prompt` 时间轴总长**必须严格等于 duration**：最后一段的结束秒数就是 duration，不许写超或留空
- 全集节奏：钩子镜头短促（6 秒），铺垫/对峙可稍长（8-10 秒），名场面给足（10-12 秒）

## 镜头衔接（连贯性）

成片是把各镜头视频**按顺序硬拼**的，相邻镜头必须首尾咬合，否则观众看到的就是跳切：

- `result` 必须写清**本镜结束瞬间的定格状态**：人物位置/姿态/表情/道具状态/镜头落点——这是下一镜的接力棒
- 下一镜的 `action` 起点必须从上一镜 `result` 的状态自然延续：位置连续、姿态连续、情绪连续，不允许凭空跳
- 同一场景的连续镜头，**光线/色调/服装/道具状态保持一致**，只改机位和动作；只有切换 scene 时才允许环境变化
- 换场景时安排**过渡镜头**衔接（空镜、甩镜、人物出画/入画、焦点转移到物件），避免两个满构图人物镜头硬切
- 每镜 `video_prompt` 开头可加一句"承接上镜：<上一镜 result 摘要>"帮助视频模型对齐起始画面

## 镜头要素（每镜必填，越具体越好）

1. **镜头标题**：3-5 字概括核心（如"拔剑出鞘"）
2. **时间**：具体时分 + 光线（如"黄昏，斜阳低照"）
3. **地点**：场景完整描述 + 空间布局 + 环境细节
4. **景别 `shot_type`**：远景/全景/中景/近景/特写 —— 景别越近，情绪越强
5. **视角 `angle`**：平视(中立) / 仰视(强势、压迫) / 俯视(弱势、渺小) / 主观视角(代入) / 侧面 / 背面
6. **运镜 `movement`**：固定 / 推(逼近情绪) / 拉(交代环境) / 摇 / 移 / 跟 / 环绕 / 手持(纪实感)，**并注明速度**：缓慢 / 平稳 / 快速 / 剧烈
7. **构图**：三分法 / 中心 / 对角线 / 前景遮挡(framing) / 纵深层次（前中后景）/ 留白
8. **光影**：硬光(戏剧、硬阴影) 或 柔光(柔和) ＋ 方向(顺光/侧光/逆光/顶光/底光) ＋ 色温(暖 2700K / 中性 / 冷 5600K) ＋ 光比(高=强反差/低=平和)
9. **景深焦距**：浅焦(f/1.8，虚化背景、突出主体) 或 深焦(f/8，环境清晰) ＋ 焦距(广角 24mm 张力夸张 / 标准 50mm 自然 / 长焦 85mm 压缩、人像感)
10. **情绪节拍**：这个镜头的戏剧张力点是什么（如"压抑后的爆发""察觉异样的瞬间"）
11. **动作 `action`**：谁 + 具体怎么做 + 肢体细节 + 表情
12. **对话 `dialogue`**：该镜头完整对话；旁白写 `旁白：内容`
13. **画面结果 `result`**：动作的即时后果 + 视觉细节
14. **氛围 `atmosphere`**：综合光线 + 色调 + 声音 + 整体情绪基调
15. **静态画面提示词 `image_prompt`**：用于首帧/尾帧/镜头图片
16. **视频提示词 `video_prompt`**：按 3 秒分段的视频生成描述（必填）
17. **配乐 `bgm_prompt` / 音效 `sound_effect`**：贴合该镜头情绪的配乐风格 / 关键环境音、动作音
18. **场景关联 `scene_id` / 角色关联 `character_ids`**

## 运镜词库（`movement` 与 `video_prompt` 的运动调色板）

按需取用。**同一序列内运动要有动静、快慢、推拉的对比**，避免通篇一种运镜；单镜运动别超过 2 种，且每条都注明速度（缓慢/平稳/快速/剧烈）。

**① 固定与微动**
- **固定 / 锁定（Static）**——机位不动、画面稳定客观；用于对话、信息交代、克制情绪。
- **呼吸感微颤（Wiggle）**——极轻微晃动如屏息；制造紧张、临场、不安。

**② 推拉（情绪的伸缩）**
- **推镜（Dolly In）**——缓慢逼近主体，情绪聚拢、代入加深。
- **超级推（Super Dolly In）**——长距离强力推进，仪式感、命运逼近。
- **急推 / 变焦猛推（Crash Zoom In）**——骤然放大，冲击、震惊、爆点。
- **拉镜（Dolly Out）**——后退拉开，交代环境、抽离、孤独、落幕。
- **急拉（Crash / Rapid Zoom Out）**——骤然拉远，失落、渺小、反差或喜剧。
- **溜溜球变焦（YoYo Zoom）**——一推一拉往复，焦灼、纠结。
- **眩晕变焦 / 移焦（Dolly Zoom，希区柯克）**——推轨与变焦反向、背景诡异伸缩；眩晕、顿悟、世界崩塌的瞬间。

**③ 摇移（横向扫视）**
- **摇镜（Pan 左/右）**——机位不动横向转，扫视环境、引出新主体。
- **俯仰（Tilt 上/下）**——纵向转：自下而上显威压/全貌，自上而下显弱势/打量。
- **横移（Truck / Dolly 左右）**——机位随主体平移，跟随行走、平滑跟踪。
- **甩镜（Whip Pan）**——快速横甩带糊影，强力转场、节奏提速。

**④ 升降吊臂（空间纵向）**
- **升镜（Crane / Jib Up）**——徐徐升起，扩展视野、释怀、宏大收尾。
- **降镜（Crane / Jib Down）**——自上而下落，聚焦、压近、揭示细节。
- **过顶（Crane Over Head）**——越过主体头顶的弧线，视点交接、空间穿越。
- **航拍拉升 / 顶视（Aerial Pullback / Overhead）**——升至高空或正俯视，上帝视角、格局、命运感。

**⑤ 环绕旋转（包裹强调）**
- **弧形环绕（Arc）**——半弧绕行主体，立体感、强调、暗流涌动。
- **360 环绕（360 Orbit）**——整圈绕行，高光时刻、强烈聚焦。
- **子弹时间（Bullet Time）**——时间凝固、机位飞速绕行；名场面、超现实爆点。
- **水平转盘（Lazy Susan）**——主体居中、背景旋转，迷离、失衡。

**⑥ 跟拍与动感（动作戏）**
- **跟拍（Follow / Head Tracking）**——锁定主体随其运动，代入、紧贴动作。
- **手持纪实（Handheld / BTS）**——自然晃动，真实、慌乱、临场感。
- **主观视角 / 第一人称（POV）**——以角色或物件之眼所见，强代入。
- **穿越机 FPV（FPV Drone）**——高速穿梭飞行，凌厉、空间炫技。
- **车载 / 追车（Car Grip / Chase / Road Rush）**——绑车机位或追逐，速度与危机。
- **急动 / 系好安全带（Buckle Up）**——骤起的猛烈运动，事态突变。
- **机械臂（Robo Arm）**——精准复杂的程控运动，科技、华丽。
- **绑身机位（Snorricam）**——机位固定于演员身上、主体稳而世界乱晃；精神失序、眩晕。

**⑦ 特殊炫技（点睛、慎用）**
- **穿物而入/出（Through Object）**——穿过钥匙孔/门缝/瞳孔等，转场、窥视、超现实。
- **眼入 / 口入（Eyes In / Mouth In）**——推进至眼或口而入，意识入口、梦境切换。
- **焦点转移（Focus Change / 移焦）**——前后景对焦切换，引导视线、揭示关系。
- **鱼眼（Fisheye）**——广角畸变，迷幻、压迫、非常态。
- **荷兰角 / 倾斜（Dutch Angle）**——地平线倾斜，失衡、危险、心理扭曲。
- **慢门拖影（Low Shutter）**——运动残影，迷醉、时间黏滞。
- **延时（Hyperlapse / Timelapse）**——时间飞逝，转场、岁月流逝。
- **英雄机位（Hero Cam）**——低角度仰拍登场，强势、高光。

## 布光词库（`lighting` / `atmosphere` / `image_prompt` 的光线调色板）

一束光由四要素定调：**质感（软/硬）＋ 方向（顺/侧/逆/顶/底）＋ 色温（暖/中/冷）＋ 光比（高反差/低平和）**（见镜头要素第 8 条）。下列是常用命名布光法，选定后直接写进提示词。

**经典人物布光**
- **伦勃朗光（Rembrandt）**——侧 45° 主光，暗侧颧骨下一块三角光；沉稳、古典、内敛。
- **蝴蝶光（Butterfly / 派拉蒙光）**——正前略高，鼻下对称蝶形影；美颜、优雅、明星感。
- **分割光（Split）**——纯侧光，半脸亮半脸暗；对立、内心撕裂、双面。
- **环形光（Loop）**——略偏的柔主光，鼻翼小环影；自然、日常、亲切。
- **宽光 / 窄光（Broad / Short）**——照亮朝向/背离镜头的半脸，分别显脸宽/显脸瘦添神秘。

**戏剧氛围**
- **低调光（Low-key）**——大面积暗、局部亮、高反差；悬疑、黑色电影、危机。
- **高调光（High-key）**——整体明亮、影淡；轻松、梦幻、纯净。
- **轮廓光 / 逆光勾边（Rim / Backlight）**——主体背后打光勾出轮廓；剪影、神圣或危险、登场。
- **顶光（Top）**——正上方下压，眼窝鼻下重影；压迫、审讯、神性。
- **底光（Underlight）**——自下而上、违反自然；诡异、恐怖、反派。
- **侧光（Side）**——纯侧打亮一侧；立体、硬朗、紧张。

**自然光线情境**
- **黄金时刻（Golden Hour）**——日出日落暖金低斜光；浪漫、怀旧、温柔。
- **蓝调时刻（Blue Hour）**——日落后冷蓝天光；孤寂、清冷、过渡。
- **正午硬光（Harsh Noon）**——头顶强硬光、短黑影；酷烈、暴露、无处可藏。
- **阴天柔光（Overcast）**——均匀漫射、无明显投影；平静、写实、阴郁。
- **窗光（Window）**——侧向柔和方向光；文艺、私密、安宁。

**实用光源（practical，入画或定调）**
- **烛火 / 篝火**——暖橙跳动光；亲密、原始、温暖中藏不安。
- **月光**——冷蓝低照；静谧、孤独、夜。
- **霓虹 / 赛博**——品红＋青的撞色；都市、迷离、欲望。
- **屏幕光**——冷白单向照脸；孤独、现代、沉迷。
- **路灯 / 车灯**——街头点状光源；夜戏、漂泊、危险街角。

## 叙事节拍 beat（短剧戏剧结构）

短剧靠"钩子—爽点—反转"驱动。分镜的运镜/景别/光线都要**服务于戏剧节拍**，而非平均用力。

**序列节拍（整段的戏剧弧）**
1. **钩子（冷开场）**——开篇 1–2 镜直给冲突/悬念/反常，**3 秒内抓住人**；常用急推特写、反常画面、强光比。
2. **建置**——交代人物关系与处境，节奏可缓；常用全景、缓移、柔光。
3. **升级**——矛盾层层加码：景别渐近、运镜渐快、光比渐强。
4. **反转 / 爆点**——情绪与信息的最高峰、名场面；常用 360 环绕/子弹时间/急推、特写、逆光勾边。
5. **钩子收尾（悬念）**——结尾留扣子勾下一集，别给完整闭环；常用拉镜抽离、定格、焦点转移到关键物件。

**单镜情绪节拍（`emotion_beat` 取值参考）**
悬念铺垫 / 情绪积累 / 察觉异样 / 对峙张力 / 情绪爆发 / 反转揭示 / 释然回落 / 留白余韵 / 高光名场面。

**节拍 → 镜头语法对照（拆解时据此从上面词库选词）**

| 节拍 | 景别 | 运镜 | 布光 |
|---|---|---|---|
| 铺垫/建置 | 全景·中景 | 固定·缓移·缓推 | 柔光·高调·窗光 |
| 积累/察觉 | 中近景 | 轻推·焦点转移 | 侧光·低调起势 |
| 对峙/张力 | 近景·特写交切 | 弧形环绕·荷兰角 | 硬光·分割光·高光比 |
| 爆发/反转 | 特写·大特写 | 急推·360环绕·甩镜·子弹时间 | 逆光勾边·顶光·强反差 |
| 回落/留白 | 全景·空镜 | 拉镜·升镜 | 黄金时刻·阴天柔光 |

## 导演级组接语法（镜头之间凭什么剪在一起）

成片是把独立生成的片段硬拼的，下列电影文法决定了硬拼是否像"剪辑"而不是"跳帧"。逐条执行：

**① 轴线规则（180°线，多人同框戏的生命线）**
- 每场戏开拍前先在心里画一条"轴线"（两人对话=两人连线；运动=运动方向线）
- 该场所有镜头的机位**必须在轴线同一侧**：A 始终在画左、B 始终在画右，绝不翻转
- 在 `image_prompt`/`video_prompt` 里**显式写出画面方位**："<role>女孩A</role>位于画面左侧，面朝画右"
- 确需越轴时，中间必须垫一个中性镜头（正面特写/空镜/沿轴线机位）再过去

**② 对话正反打（对话戏的标准拍法）**
- 对话序列结构：**定场镜头**（全景/中景交代两人位置关系）→ **过肩正反打交替**（拍 A 时带 B 的肩/后脑勺作前景）→ 情绪顶点给**单人特写**
- 正反打两个机位与轴线的夹角对称、景别一致、视线高度一致，来回切才像同一场戏
- 谁说话拍谁是基本盘；关键反应可以拍听者（反应镜头）

**③ 30° 规则（防跳帧感）**
- 同一主体的相邻镜头，机位角度变化 ≥30°，或景别至少跳一档（中景→特写）
- 只变角度不变景别、或只变景别不变角度都行，**两者都不变=跳帧**，禁止

**④ 动作剪辑点（硬拼救星）**
- 尽量把镜头边界切在**动作进行中**：上一镜结束于"手伸向门把手"，下一镜开始于"门被推开"
- 即 `result` 写"动作到一半的瞬间"，下一镜 `action` 写"该动作的完成"——观众的眼睛会自动缝合
- 避免每镜都从静止开始、到静止结束（那是 PPT 不是电影）

**⑤ 视线与运动方向匹配**
- 人物看画右，下一镜他看的东西就该出现在（他的）画右方向
- 人物向画右出画，下一镜必须从画左入画（方向一致）；反向 = 观众以为他折返了
- 追逐/行进戏全程保持同一运动方向，所有镜头统一写"向画右移动"

**⑥ 全集光色弧线（色调服务情绪推进）**
- 按叙事节拍给整集设计色温走向，写进各镜 `atmosphere`：如 建置(中性日光)→积累(转阴/侧光)→对峙(冷蓝/高反差)→爆发(强逆光/撞色)→收尾(暖金或抽色)
- 同一场景内色调只能渐变不能突变；突变只发生在场景/情绪转折点上

**⑦ 主观视角链（POV 三件套）**
- "谁在看"的戏用三镜组：**人物看（近景）→ 他看到的（POV）→ 他的反应（特写）**，顺序不可乱
- POV 镜头的机位高度/角度 = 该角色的眼睛位置

## 角色一致性（双重约束法 —— 重中之重）

漫剧/短剧最容易崩的就是角色一致性。规则：

- **凡有角色出场的镜头，`image_prompt`/`video_prompt` 必须原封带入该角色的"特征锚点"**——从 `read_storyboard_context` 返回的角色 `appearance` 里提取并保留：**发型/发色、五官标志(如疤、痣)、服装版型与配色、头饰/配饰、妆容**。
- 换场景、换光线、换动作时，这些特征关键词**一字不改地重复**，只改动作/环境/光线。
- 例：角色「沈墨」= `墨色长衫、束发玉冠、左眉旧疤、清冷眉眼`。无论哪个镜头，只要他出场，这串锚点就照抄进提示词。
- 视频生成支持参考图绑定（@角色图），锚点文字 + 参考图**双保险**。

## 视频提示词格式

```
0-3秒：<location>万佛宗大殿前</location>，中景略仰视，<role>沈墨[墨色长衫·束发玉冠·左眉疤]</role>缓缓拔剑，眼神冷冽。镜头缓慢推近。侧逆光、暖金色温、浅焦虚化背景。
<n>3-6秒：<location>万佛宗大殿前</location>，特写，剑锋寒光映出<role>沈墨</role>半张脸，嘴角微扬。固定镜头。
```

标签：`<location>地点</location>` `<role>角色名[特征锚点]</role>` `<voice>旁白</voice>` `<n>`时间段分隔

## 使用步骤

1. 调用 `read_storyboard_context` 读取剧本、角色（含 appearance）、场景、已有分镜
2. 基于剧本完成拆解：先按"时长适配"分配每镜秒数（只取 6/8/10/12），再按"镜头衔接"让相邻镜头首尾咬合，然后逐场检查"导演级组接语法"（轴线/正反打/30°规则/动作剪辑点/方向匹配），同时保证镜头语言有节奏变化（不要全是中景平视）
3. 为每镜补全全部字段（电影维度务必填：`shot_type`景别 / `angle`视角 / `movement`运镜 / `lighting`布光 / `composition`构图 / `emotion_beat`情绪节拍），**有角色就带特征锚点**
4. 调用 `save_storyboards` 一次性保存；如需调整用 `update_storyboard`

## 场景/角色绑定规则

- 优先用 `read_storyboard_context` 返回的 `scenes`；`location+time` 能匹配时必须回填正确 `scene_id`，不要凭空造场景 ID
- `character_ids` 从返回的角色列表选；镜头里明确出场/被看见/动作/说话的角色都要绑；纯环境/空镜/物件镜头传空数组

## 质量要求

- 镜头语言要**有变化、有节奏**：远近交替、视角高低错落、运镜动静结合——避免通篇"中景·平视·固定"
- `description` 给人读，`video_prompt` 给模型生成，二者不互相替代
- `image_prompt` 突出单帧构图、角色外观锚点、环境与光线
- `video_prompt` 突出时间推进、动作变化、镜头运动
- `bgm_prompt`/`sound_effect` 用具体短语，别空泛到只剩"紧张""悲伤"
- **任何有角色的提示词，先检查特征锚点带了没**

# 剧本润色

## 任务说明

用户提供初稿剧本，你需要按短剧爆款标准润色优化。**不改变核心剧情**，只在叙事质感、节奏、台词上做提升。

## 润色四维度

### 1. 节奏诊断
- 找出"水戏"（无信息量的过渡）→ 删
- 找出关键冲突点 → 强化
- 每场戏控制在 15-25 秒，删掉拖沓部分

### 2. 冲突强化
- 每场戏必须有**目标 + 阻碍 + 选择**
- 主角必须有"想要 vs 必须"的内心拉扯
- 反派要有合理动机（不是工具人）
- 升级冲突：每集比前一集更紧张

### 3. 人物塑造
- 每个角色第一次出场要有**记忆点**（口头禅、标志动作、外貌特征）
- 主角不要过于完美（要有缺点才真实）
- 配角不要工具化（每人有自己的小目标）

### 4. 对白优化
- 每句话必须**推进剧情**或**展现性格**
- 删除"嗯""啊""好的"等无意义对白
- 用动作代替形容词（"愤怒" → 不写，让角色摔杯子）
- 金句：每集至少 1 句可截图传播的话
- 短剧台词要口语化，避免书面语

## 工作流程

1. 通读用户剧本，标记问题（开场弱 / 节奏拖 / 对白平淡 / 人物扁平）
2. 逐场润色，保留剧情走向
3. 输出优化后的完整剧本 + 关键改动说明

## 输出格式

```json
{
  "polished_script": "...完整润色后剧本...",
  "key_changes": [
    "S01 开场：原版花 30 秒铺垫，改为 5 秒强冲突切入",
    "S03 对白：林深原版"我很生气"改为摔杯动作 + "你说话啊！"",
    "新增 S04 末尾钩子：方泉留下神秘字条"
  ],
  "rating": {
    "node_score": 8.5,
    "conflict_score": 9.0,
    "dialogue_score": 8.0,
    "overall": 8.5,
    "suggestion": "建议增加 1 个情感转折点"
  }
}
```

## 常见问题诊断对照

| 症状 | 原因 | 修正 |
|---|---|---|
| 开场无吸引力 | 慢热铺垫 | 5 秒内必须有钩子（冲突/悬念/反转）|
| 中段拖沓 | 水戏过多 | 删除无信息推进的场景 |
| 对白生硬 | 书面语 | 口语化 + 加动作 |
| 角色平 | 工具人 | 给每个角色一个独特动机 |
| 结局突兀 | 缺铺垫 | 中段插入伏笔 |
| 钩子弱 | 句号收尾 | 改成悬念 / 突发 / 神秘人物登场 |

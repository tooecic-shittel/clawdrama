# 角色与场景提取指南

## 角色提取规范

为每个角色建立「**角色资产卡**」——后续所有镜头都靠它锁一致性，所以外貌必须**结构化、可复用、特征鲜明**（专业漫剧的「角色资产标准化」做法）。

- **姓名**：角色全名
- **角色定位**：主角/配角/龙套
- **外貌描写 `appearance`**（核心，必须结构化，逐项给具体值，别泛泛）：
  - **性别 / 年龄段 / 体型身高**
  - **发型 + 发色**：具体到样式与颜色（如"齐肩乌黑直发、左侧编一缕"）
  - **五官 + 标志性特征**：越独特越锁得住（如"丹凤眼、左眉一道旧疤、唇下一颗痣"）—— 避免"大众脸"，每个角色至少 1–2 个**独有记忆点**
  - **服装版型 + 配色**：版型和颜色都要定（如"墨色立领长衫、银线暗纹、束腰"）
  - **配饰 / 随身物**：如"束发玉冠、腰间长剑、右手玉扳指"
  - **气质 / 神态**：如"清冷孤傲、眼神锐利"
  - 若剧本**没写明**，要**主动设计**一套**独特且固定**的形象，并全程保持一致，不得在不同集/镜头里漂移
  - 最后把上述关键特征浓缩成一串**「锚点关键词」**放在 `appearance` 开头，供分镜与角色图**原封复用**，例：`墨色立领长衫·束发玉冠·左眉旧疤·丹凤眼·清冷孤傲`
- **性格特点**：核心性格标签
- **角色描述**：背景故事和关系

## 场景提取规范

场景是镜头的「舞台」，提示词要有电影感的空间、光影与氛围（**纯背景、不含人物**），并保持全剧画风一致。

- **地点 `location`**：具体场所 + 空间布局（如"万佛宗大殿前石阶广场，两侧立柱、远处殿宇层叠"）
- **时间 `time`**：时间段 + 光线条件（如"黄昏，斜阳低照、长影"）
- **氛围 `atmosphere`**：环境情绪基调 + 主色调（如"肃穆压抑、青灰冷调"）
- **图片提示词 `prompt`**（英文、纯背景、结构化）：
  - **空间纵深**：前景 / 中景 / 远景的层次与引导线
  - **光影**：光质(硬光/柔光) + 方向(顺/侧/逆/顶光) + 色温(暖 2700K / 冷 5600K)
  - **色调 / 调色板**：整体配色倾向
  - **材质细节**：石质、木纹、雾气、水面反光、织物等具体质感
  - **风格收尾**：cinematic、与全剧画风一致、high quality、no people、no text

## 道具提取规范

提取的道具信息包含：
- **名称**：道具名
- **类型**：日常/武器/交通/装饰等
- **描述**：外观和用途
- **图片提示词**：用于AI图片生成的英文提示词

## 使用步骤

1. 调用 `read_script_for_extraction` 读取当前集剧本
2. 调用 `read_existing_characters` 查看项目已有角色和当前集已关联角色
3. 调用 `read_existing_scenes` 查看项目已有场景和当前集已关联场景
4. 只提取当前集真实涉及的角色和场景
5. 调用 `save_dedup_characters` 保存角色并自动关联到当前集
6. 调用 `save_dedup_scenes` 保存场景并自动关联到当前集

## 当前集规则

- 目标是补齐“当前集”需要的角色和场景，不是重扫整个项目
- 若角色或场景已在项目中存在但当前集未关联，仍应复用并关联到当前集
- 若项目中已有同名角色或同地点同时间场景，优先复用，不要重复创建
